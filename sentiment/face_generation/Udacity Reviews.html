TIPS:

  • Add dropout to prevent the discriminator from overfitting and making mistakes. To implement it, use them after you activation functions from the conv2d() layers. Remember, since you are using Tensorflow layers, make sure to use the layers version of dropout. Also, make sure to set the training=is_train parameter on the generator and to start off with a small rate. Tensorflow’s tf.layers.dropout() has the rate parameter and not keep_prob, make sure you are using the right values when training because they are not the same thing. here is a simple equation keep_prob = 1 - rate. To learn more and credit for the equation are in this link: What's the difference between tf.nn.dropout and tf.layers.dropout A possible implementation is as follows:

Generator:
tf.layers.dropout(x, rate, training=is_train)

Discriminator:
tf.layers.dropout(x, rate)

  • A good coding practice is to make your code modular and easy to read. An example to accomplish this is to make a separate function called leaky_relu() and add the appropriate parameters. A possible implementation is as follows:
    def leaky_relu(x, alpha=0.05, name='leaky_relu'): 
   return tf.maximum(x, alpha * x, name=name)
  • flat = tf.contrib.layers.flatten(x3) works fine but flat = tf.reshape(x3, (-1, 4 * 4 * 256)) helps improve the stability between the discriminator and the ​generator.

NOTE:

  • Remove all np.random.uniform() and instead use the suggestion below. The np.random.uniform() adds unnecessary​ noise that negatively affects the GAN model.
  • To prevent the discriminatorfrom becoming too strong and to help it generalize better, the discriminator labels can be reduced from 1 to 0.9. This is called label smoothing (one-sided).
    A possible TensorFlow implementation is:
smooth = 0.1
d_loss_real = tf.reduce_mean(
        tf.nn.sigmoid_cross_entropy_with_logits(logits=d_logits_real, labels=tf.ones_like(d_model_real)*(1.0 - smooth)))

A possible TensorFlow implementation is tf.ones_like(tensor) * (1.0 - smooth) where smooth is a float value ~ 0.1. The resulting value will multiply the label by ~0.9 and reduce the label from 1 to ~0.9.
