You should use tf.variable_scope with the reuse_variables() function because otherwise you make more than one context manager unnecessarily. In addition, thanks to this function you will be able to have the same indent block. Therefore please replace your line:

with tf.variable_scope("decoding", reuse=True) as decoding_scope:

with this one:

decoding_scope.reuse_variables()

and indent this line properly to be aligned the same way as the line above it and below it.

In addition, please delete the second occurrence of your line:

with tf.variable_scope('decoding', reuse=None) as decoding_scope:

because you don't need it.
